## Featured Projects

### 🗂️ MRF Digitalization System *(Internship)*
> React 18 · TypeScript · Tailwind CSS · Firebase · Cloudinary

A full digital replacement for paper-based Material Request Forms at HS Technologies. Features real-time data sync via Firestore, Firebase Auth, Cloudinary media uploads, and a multi-step approval/denial/revision workflow. Built and maintained with branch-based Git workflows alongside a backend collaborator.

---

### 🔐 PhiNex — Phishing Detection Dashboard
> React 18 · MUI v5 · Socket.IO · Chart.js

Frontend monitoring dashboard for a PYNQ-Z2 ARM-FPGA phishing detection system. Streams real-time threat data via Socket.IO and visualizes network analytics with Chart.js — bridging embedded hardware and modern web UI.
