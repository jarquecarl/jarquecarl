## Featured Projects
 
### 🗂️ MRF Digitalization System *(Internship)*
> React 18 · TypeScript · Tailwind CSS · Firebase · Cloudinary
 
A full digital replacement for paper-based Material Request Forms at HS Technologies. Features real-time data sync via Firestore, Firebase Auth, Cloudinary media uploads, and a multi-step approval/denial/revision workflow. Built and maintained with branch-based Git workflows alongside a backend collaborator.
 
---
 
### 🔐 PhiNex — Phishing Detection Dashboard
> React 18 · MUI v5 · Socket.IO · Chart.js
 
Frontend monitoring dashboard for a PYNQ-Z2 ARM-FPGA phishing detection system. Streams real-time threat data via Socket.IO and visualizes network analytics with Chart.js — bridging embedded hardware and modern web UI.
 
---
 
### 💸 Fundo — Expense Tracker
> React · JavaScript · Recharts · Tailwind CSS · Vite · Firebase
 
Personal finance tracker with category breakdowns, monthly summaries, and interactive Recharts visualizations. Navy + cyan dark theme with glassmorphism UI.
 
🔗 [Live Demo](https://fundo-expense-tracking-app.netlify.app/) · [Repo](https://github.com/jarquecarl-debug/fundo-expense-tracking-app)
 
---
 
### 🌀 Maze Runner — Browser Game
> TypeScript · React Three Fiber · Three.js
 
First-person 3D maze game with collectibles, an animated black hole exit portal, a hologram-shader wraith enemy, and crystal spike obstacles — built outside work to sharpen game-loop architecture and 3D rendering skills.
 
🔗 [Live Demo](https://maze-game-3d.netlify.app/) · [Repo](https://github.com/jarquecarl/maze-game)
